## Directory structure

Each platform's samples follow a consistent layout:

```
SDK/
├── Linux/                    # or OSX/, Win/
│   ├── bin/                  # Libraries and AI models
│   │   ├── libPerfectlyClearPro.a
│   │   ├── pro_and_universal_20211221.pnn
│   │   └── ...
│   ├── header/               # C/C++ headers
│   ├── presets/              # Sample presets
│   ├── SimpleAiSample/
│   ├── PresetSample/
│   └── ...
├── CLI/                      # Pre-built CLI tool
│   ├── Linux/
│   ├── Win/
│   └── OSX/
├── sdk_license/              # License files
└── presets/                  # Global presets
```

## Quick start

### Build a C/C++ sample (Linux/macOS)

```bash
cd Linux/SimpleAiSample
make -f makefile.linux
./PerfectlyClear input.jpg output.jpg -v
```

### Build a C# sample (Windows)

```bash
cd Win/CSharp_SimpleAiSample
dotnet build
dotnet run -- input.jpg output.jpg -v
```

### Use the CLI tool

```bash
export PFC_SDK_LICENSE=/path/to/sdk_license
./pfccmd input.jpg -o output.jpg
```

### Use the Container

```bash
docker load --input pfc_container_image.tar
docker run -d -v "$(pwd)/sdk_license":/sdk_license -p 80:80 pfc_container

# Upload and correct
imageKey=$(curl -H "Content-Type: image/jpeg" -X PUT "http://localhost:80/pfc" --upload-file input.jpg | jq -r '.imageKey')
curl "http://localhost:80/pfc/${imageKey}" | jq -r '.corrected_url'
```
