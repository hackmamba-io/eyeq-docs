# Windows Examples

Sample source code for applications ready to build on Windows systems. These examples show different usage models and provide a starting point for production applications.

## Simple AI Sample - Using Scene Detection

The simplest way to use Perfectly Clear AI on Windows.

### 1. Setup License Protection

```cpp
// Set the path to the license files
PFC_SetProtectionPath("C:\\path\\to\\sdk_license");
```

### 2. Load Image to Correct

```cpp
PFCImageFile imageFile;
imageFile.LoadImage("C:\\path\\to\\input.jpg");
PFCIMAGE image = imageFile.GetPFCImage();
```

### 3. Initialize the AI Engines

```cpp
PFCENGINE engine;
PFC_CreateEngine(&engine, PFCENGINE_AI);
PFC_LoadAIModels(engine, "C:\\path\\to\\models\\");
```

### 4. Perform Full Correction

```cpp
PFCPARAM param;
PFC_ReadPresets(param, "C:\\path\\to\\preset.preset");
PFC_AutoCorrect(engine, &image, NULL, &param, CALC_ALL, false, NULL);
```

### 5. Save the Corrected Image

```cpp
imageFile.SaveImage("C:\\path\\to\\output.jpg", 92);
```

## Building with Visual Studio

### Project Setup

1. Open Visual Studio
2. Create a new C++ Console Application
3. Add include directories: `Project > Properties > C/C++ > Additional Include Directories`
4. Add library directories: `Project > Properties > Linker > Additional Library Directories`
5. Add dependencies: `PerfectlyClearPro.lib`

### Required DLLs

Copy these DLLs to your output directory:

- `PerfectlyClearPro.dll`
- `PFCImageFile.dll`
- AI model files (`.pnn`)

## Detailed AI Sample - Custom Presets

To extend the simple example, you can load custom AI Presets:

```cpp
// Load custom Scene Detection presets
PFC_LoadScenePresets(engine, "C:\\path\\to\\custom_scenes.preset");
```

## Image Resizing

Load, correct, and resize images:

```cpp
// After loading and correcting the image
int newWidth = 1920;
int newHeight = 1080;

PFCImageFile resizedFile;
resizedFile.CreateResized(imageFile, newWidth, newHeight);
resizedFile.SaveImage("C:\\path\\to\\resized_output.jpg", 92);
```
