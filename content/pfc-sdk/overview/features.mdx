## Perfectly Clear feature set

The Perfectly Clear SDK provides a standard set of image enhancement features organized into several categories.

The following table summarizes the Perfectly Clear feature set. Features are described in detail after the table.
| Feature Category | Included Features | Description |
|------------------|------------------|-------------|
| Correction engine | Exposure correction, Color correction, Noise reduction, Sharpness | Automatically analyzes and corrects core image quality issues, ensuring balanced lighting, natural colors, reduced noise, and improved clarity. |
| Retouching | Skin smoothing, Pore reduction, Blemish removal, Skin tone evening | Enhances skin appearance by smoothing texture, reducing imperfections, and producing even, natural-looking skin tones. |
| Eye enhancement | Eye brightening, Red-eye removal, Eye sharpening, Catchlight enhancement | Improves eye clarity and visual appeal by brightening, sharpening details, removing red-eye, and enhancing natural sparkle. |
| Face shaping | Face slimming, Teeth whitening, Lip enhancement | Applies subtle facial adjustments to refine shape, brighten smiles, and enhance lip color and definition. |
| Detection | Multi-face detection, 68-point facial landmarks, Blink and smile detection, Profile detection | Detects and analyzes faces within an image to enable accurate, targeted corrections, even for non-frontal faces. |
| Presets | Built-in presets, Custom presets, LOOKs (creative filters) | Provides ready-made and customizable correction profiles, including artistic LOOKs for creative styling. |
| Batch processing | Multi-threading, Queue management, Progress callbacks, Error handling, Memory optimization | Enables efficient large-scale image processing with performance-focused execution and fault tolerance. |
| Platform availability | Windows, macOS, Linux, iOS, Android | Broad platform support for consistent image enhancement across desktop, server, and mobile environments. |
| AI features | Scene detection, Intelligent Auto, Skin tone detection | AI-driven analysis automatically identifies scenes, selects optimal corrections, and applies natural-looking skin tone adjustments. |
