# Changelog

## Current Version

The Perfectly Clear Web API is available at:

```
https://api.perfectlyclear.io/v1
```

## Recent Updates

### December 2025

- Added background removal feature
- Added glasses glare removal feature
- Improved AI scene detection accuracy
- Added WebP output format support

### November 2025

- Added batch processing endpoint
- Added webhook notifications
- Increased rate limits for Pro tier
- Added custom presets for Enterprise

### October 2025

- Initial public release
- Single image correction endpoint
- Preset management
- Python and Node.js SDKs

## API Versioning

The API uses URL-based versioning:

- Current: `v1`
- All breaking changes will increment the version
- Previous versions supported for 12 months after deprecation
