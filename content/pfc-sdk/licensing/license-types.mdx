# License Types

Perfectly Clear SDK offers several licensing options to fit different use cases.

## License Options

### Development License

**For:** Building and testing your integration

- ✅ Full SDK functionality
- ✅ Unlimited local testing
- ⚠️ Watermarked output
- ⚠️ Not for production use

```cpp
// Development license adds watermark to output
PFC_SetLicense(engine, "DEV-XXXX-XXXX-XXXX");
```

### Production License

**For:** Commercial deployment

- ✅ Full SDK functionality
- ✅ No watermarks
- ✅ Production use permitted
- Volume-based pricing

### Enterprise License

**For:** Large-scale deployments

- ✅ Everything in Production
- ✅ Unlimited volume
- ✅ Priority support
- ✅ Custom presets
- ✅ Offline activation

## License Comparison

| Feature | Development | Production | Enterprise |
|---------|-------------|------------|------------|
| Full features | ✅ | ✅ | ✅ |
| No watermark | ❌ | ✅ | ✅ |
| Production use | ❌ | ✅ | ✅ |
| Volume limits | N/A | Per agreement | Unlimited |
| Offline mode | ❌ | ❌ | ✅ |
| Priority support | ❌ | ❌ | ✅ |
| Custom presets | ❌ | ❌ | ✅ |

## Pricing

Contact [sales@eyeq.com](mailto:sales@eyeq.com) for pricing information.

Pricing is typically based on:

- **Per-image processing** - Pay per image corrected
- **Monthly/Annual subscription** - Fixed fee for volume
- **Perpetual license** - One-time purchase

## Getting a License

1. Visit [eyeq.com/sdk](https://eyeq.com/sdk)
2. Request a development license (free for evaluation)
3. Contact sales for production licensing
