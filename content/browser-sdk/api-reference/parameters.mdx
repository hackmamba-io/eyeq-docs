# Parameters Reference

The `PFCPARAM` structure contains all correction parameters organized into sub-structures.

## PFCPARAM

Top-level parameter container.

```typescript
interface PFCPARAM {
  core: PFCCOREPARAM;   // Core corrections
  fb: PFCFBPARAM;       // Face beautification
  nr: PFCNRPARAM;       // Noise reduction
  re: PFCREPARAM;       // Red-eye removal
  v3: PFCV3PARAM;       // V3 advanced params
  dp2: DP2PARAM;        // DP2 params
  
  // Methods
  getParamBuffer(): any;
  setOutputLUT(guid: string): void;
  getLayerProperty(layer: number, property: string): any;
  setLayerProperty(layer: number, property: string, value: any): void;
  getMaskProperty(layer: number, property: string): any;
  setMaskProperty(layer: number, property: string, value: any): void;
  getSelectiveColor(index: number, property: string): any;
  setSelectiveColor(index: number, property: string, value: number): void;
}
```
