# System Requirements

## iOS Requirements

| Requirement | Minimum | Recommended |
|-------------|---------|-------------|
| iOS Version | 13.0 | 15.0+ |
| Xcode | 14.0 | 15.0+ |
| Swift | 5.0 | 5.9+ |
| Device | iPhone 8 | iPhone 12+ |
| Architecture | arm64 | arm64 |

### Supported Devices

- iPhone 8 and later
- iPad Pro (2018 and later)
- iPad Air (3rd generation and later)
- iPad (8th generation and later)

## Android Requirements

| Requirement | Minimum | Recommended |
|-------------|---------|-------------|
| Android Version | 7.0 (API 24) | 11.0+ (API 30) |
| Android Studio | Arctic Fox | Hedgehog+ |
| Kotlin | 1.6 | 1.9+ |
| Java | 8 | 11+ |
| Architecture | arm64-v8a | arm64-v8a |

### Hardware Requirements

- **CPU**: ARMv8-A with NEON support
- **RAM**: 2GB minimum, 4GB recommended
- **GPU**: OpenGL ES 3.0+ support

### Recommended Chipsets

- Qualcomm Snapdragon 700 series or higher
- Samsung Exynos 9810 or higher
- MediaTek Dimensity 700 or higher

## AI Model Files

The SDK requires the following model files:

| File | Size | Purpose |
|------|------|---------|
| `dynamic.pnn` | ~5MB | Core correction model |
| `scene_detection.pnn` | ~2MB | Scene classification |

These files are included in the SDK package and must be bundled with your application.

## Network Requirements

- **Offline Operation**: The SDK works completely offline after initial setup
- **License Validation**: Requires one-time network access for license activation
- **Updates**: Optional network access for model updates

## Storage Requirements

| Component | Size |
|-----------|------|
| SDK Library | ~10MB |
| AI Models | ~7MB |
| Runtime Cache | ~20MB |
| **Total** | **~37MB** |
