# Enumerations

Complete reference for all enums available in the Browser SDK.

## InferenceOption

AI inference mode selection.

```typescript
enum InferenceOption {
  local = 0,        // Process in browser
  remote = 1,       // Process on server
  remoteSocket = 2  // Legacy WebSocket
}
```

## PFCFEATURE

Feature flags for calculation.

```typescript
enum PFCFEATURE {
  CALC_CORE = 1,              // Core corrections
  CALC_NR = 2,                // Noise reduction
  CALC_FB = 4,                // Face beautification
  CALC_RE = 8,                // Red-eye removal
  CALC_ALL = 15,              // All classic features
  CALC_NOTINTCALC = 16,       // Skip tint calculation
  CALC_NOFAE = 32,            // Skip FAE
  CALC_FAEHISPEED = 64,       // High-speed FAE
  CALC_CARTOON_DETECTOR = 128,// Cartoon detection
  CALC_SHARP = 256,           // Sharpness analysis
  CALC_SCENE_DETECTION = 512, // AI scene detection
  CALC_COMPOSITE = 1024,      // Composite detection
  CALC_SKINTONE = 2048,       // Skin tone detection
  CALC_AICOLOR = 4096,        // AI white balance
  CALC_DYNAMIC = 65536        // Dynamic corrections
}
```

**Usage:**
```javascript
const features = PFCFEATURE.CALC_ALL | PFCFEATURE.CALC_NOTINTCALC;
const aiFeatures = PFCFEATURE.CALC_SCENE_DETECTION | PFCFEATURE.CALC_SKINTONE;
```

## PFCAPPLYSTATUS

Return status from apply operations.

```typescript
enum PFCAPPLYSTATUS {
  APPLY_SUCCESS = 0,                    // Success
  APPLY_ERROR_PROFILE_MISSING = -1,     // No profile
  APPLY_ERROR_ENGINE_MISSING = -2,      // No engine
  APPLY_CANCELLED = -3,                 // Operation cancelled
  APPLY_NOSOURCE = -4,                  // No source image
  APPLY_BADFORMAT = -5,                 // Bad image format
  APPLY_INVALIDLICENSE = -6,            // License invalid
  APPLY_WASM_INVALID_DOMAIN = -7,       // Domain not allowed
  APPLY_WASM_INVALID_APIKEY = -8,       // Invalid API key
  APPLY_WASM_INVALID_CERTIFICATE = -9,  // Certificate invalid
  APPLY_WASM_EXPIRED_CERTIFICATE = -10, // Certificate expired
  APPLY_WASM_CERTIFICATE_WRONG_VERSION = -11, // Version mismatch
  APPLY_LOOKS_MISSING = -12,            // LOOKs not loaded
  APPLY_ERROR_AI_CORRECTION_MISSING = -13 // AI data missing
}
```
